--- title : "windows" ---

Falls Sie Windows Home als Ihr Betriebssystem für Ihren Knoten benutzen, werden Sie Probleme mit dem Ausführen des Docker Images bekommen. Windows Home Benutzer müssen noch auf ein späteren Binary Update warten, bevor sie dem V3 Netzwerk beitreten können, oder sie sollten auf ein anderes Betriebssystem überwechseln. Falls Sie auf den Binary Update warten möchten, bevor Sie unserem Netz beitreten, heben Sie sich bitte den Authorizationstoken auf und wir werden alle Speicherknotenbetreiber-Eingeladenen benachrichtigen, sobald dieses Problem behoben ist
