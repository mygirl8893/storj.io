На этой стадии Альфа тестирования Storj Labs будет контролировать поток новых данных, загружаемых в сеть. Эти данные будут использоваться с целью тестирования для подготовки к выпуску версии Vanguard, который откроет сеть для клиентов и партнёров из списка ожидания для разработчиков.
